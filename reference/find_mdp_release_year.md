# Find the MD Planning release year for a given data year

The MD Planning releases annual reports that contain 10+ years of
historical data. This function determines which release year contains
data for the requested end_year.

## Usage

``` r
find_mdp_release_year(end_year)
```

## Arguments

- end_year:

  The school year end to find data for

## Value

The release year, or NULL if not available
