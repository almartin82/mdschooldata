# Download enrollment data from Maryland Department of Planning

Downloads historical enrollment data from the Maryland Department of
Planning's public school enrollment projections. This provides
enrollment by grade level for all 24 Maryland jurisdictions from
2014-present.

## Usage

``` r
download_mdp_enrollment(end_year)
```

## Arguments

- end_year:

  School year end (e.g., 2024 for 2023-24)

## Value

Data frame with enrollment by jurisdiction and grade
