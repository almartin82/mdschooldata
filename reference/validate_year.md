# Validate end_year parameter

Validate end_year parameter

## Usage

``` r
validate_year(end_year, min_year = 2003, max_year = NULL)
```

## Arguments

- end_year:

  Year to validate

- min_year:

  Minimum valid year (default 2003)

- max_year:

  Maximum valid year (default current year + 1)

## Value

TRUE if valid, throws error otherwise
