# Extract grade-level enrollment for a jurisdiction

Extract grade-level enrollment for a jurisdiction

## Usage

``` r
extract_jurisdiction_grades(df, j_row, year_col, end_year)
```

## Arguments

- df:

  The Excel data frame

- j_row:

  Row number of the jurisdiction header

- year_col:

  Column number containing the target year's data

- end_year:

  The school year end

## Value

Data frame with one row containing grade-level enrollment
